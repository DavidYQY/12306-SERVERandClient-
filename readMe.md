## C++大作业——12306++铁路票务系统
### 前言
这是清华大学自动化系c++小学期的大作业，是命题形式给出，大致就是做出现有12306软件的全部功能。大作业要求见:"C++程序设计与训练....docx"

### 12306++铁路票务系统简介
分为服务端和客户端。
服务端负责数据库管理和逻辑操作
客户端负责用户UI和其他逻辑

### 开发环境
VS2012+Qt5.1.0
MinGW+Qt5.7.0
数据库：SQLite
网络通信：TcpSocket
